# semantic-tv
- Chuẩn bị dữ liệu theo tên sample.mp4, sample-meta.txt, sample-uid.txt
	+ sample.mp4: video
	+ sample-meta: mỗi dòng
		mm:ss	x y w h	id
	+ sample-uid.txt: mỗi dòng là tên người ứng với id 0, 1, 2...
- Khởi động app
- Open để mở video:
	+ Move farward để tiến
	+ Move back để lù
	+ Click vào khuôn mặt để xem ID
